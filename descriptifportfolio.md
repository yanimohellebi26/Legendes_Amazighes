# Légendes Amazighes — Interactive 3D Mythology Game

## Tagline
A Unity 3D game exploring Amazigh (Berber) myths and legends through interactive storytelling.

## Description
An interactive 3D game built in Unity that brings Amazigh (Berber) mythology to life. Players explore a hub world and select myths to experience through interactive storytelling. Features a ScriptableObject-based data architecture for stories, heroes, and dialogues, with a state-machine game manager and progression tracking.

## Motivation
Cultural preservation through interactive media — making ancient Amazigh myths accessible and engaging for modern audiences through 3D game experiences.

## Tech Stack
- **Engine**: Unity (C#)
- **Architecture**: ScriptableObjects, State Machine (GameManager)
- **Patterns**: Singleton, Hub World, Scene Management
- **Data**: ScriptableObject-based story/hero/dialogue system

## Key Features
- GameManager singleton with state machine (MainMenu, Hub, PlayingLevel, Paused)
- ScriptableObject-based story, hero, and dialogue data architecture
- Hub world pattern — central area to select myths to experience
- Player controller with story interactable system
- Progression tracking (completed/unlocked stories)
